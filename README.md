## Installation
This package is designed to be run in a Node environment. For the best developer experience use Typescript.
```sh
$ npm i ping-rx
```

## CLI
For CLI usage its easier to install globally like so:
```sh
$ npm i -g ping-rx

// starting listening
$ pingrx -h 192.168.1.73
```
